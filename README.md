# html-fireword

Interactive fireworks animation library for HTML canvas.

## Description

html-fireword is a lightweight JavaScript library that creates beautiful, customizable firework animations on HTML canvas elements. Perfect for celebrations, interactive experiences, and adding visual flair to your web applications.

## Installation

```bash
npm install html-fireword
```

## Usage

```javascript
import Firework from 'html-fireword';

// Create a new firework instance
const firework = new Firework({
  // Configuration options
});

// Launch a firework at specific coordinates
firework.launch(x, y);

// Stop the animation
firework.stop();
```
